# Sistema Simples de Cadastro de Mensagens

Um sistema web básico e responsivo para envio de recados, focado no estudo da integração entre Front-end e Back-end (PHP).

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML:** Estrutura semântica do formulário.
* **CSS:** Estilização externa com foco em design responsivo (Flexbox).
* **JavaScript:** Validação de campos no lado do cliente (Client-side) para evitar envios vazios.
* **PHP:** Processamento de dados no servidor e proteção básica contra scripts maliciosos.

## Como rodar o projeto localmente

Para executar este projeto, você precisará de um servidor local que suporte PHP (como o XAMPP).

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```
2.  **Mova a pasta para o servidor:**
    Coloque a pasta do projeto dentro do diretório `htdocs` do seu XAMPP.
    *Caminho comum: `C:\xampp\htdocs\CadastroMensagens`*

3.  **Inicie o Servidor:**
    Abra o Painel de Controle do XAMPP e clique em **Start** no módulo **Apache**.

4.  **Acesse no Navegador:**
    Digite o seguinte endereço na barra de busca:
    `http://localhost/CadastroMensagens/index.php`

## Responsividade

O design foi construído para ser "Mobile First", garantindo que o formulário seja fácil de usar tanto em dispositivos móveis quanto em computadores, utilizando unidades relativas e @media queries.
